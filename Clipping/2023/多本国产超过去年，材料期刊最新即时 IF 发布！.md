为了对即时 IF 与 2021 正式 IF 进行对比，我们**原创性地提出了 “****完成度****” 的概念**。完成度指即时 IF 达到了上年度正式 IF 的百分之多少，其实就是即时 IF 与 2021IF 的比值，可以判断每本期刊的 IF 增速情况。

本期发布**材料期****刊（共计 422 本）**的最新即时 IF，更多期刊情况，可于**学之策公众号后台查询**。
